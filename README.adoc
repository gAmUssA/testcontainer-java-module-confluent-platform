== Getting binaries

NOTE: While library is `0.1-SNAPSHOT` binaries hosted in jitpack

[source,groovy]
----

  repositories {
    maven { url 'https://jitpack.io' }
  }
  dependencies {
    implementation 'com.github.gAmUssA:testcontainers-java-module-confluent-platform:master-SNAPSHOT'
  }
}
----
